Sample_wraped_text v2

Sample wraped text for MRE platform (including Nokia S30+). Key "Left soft key" open input dialog, key "Right soft key" close program. Tested on Nokia 225 with display resolution 240x320. For using with Nokia mobile phone, app must be signed with IMSI (your SIM card) code. https://vxpatch.luxferre.top. Application files - "Sample_wraped_text.vxp"
